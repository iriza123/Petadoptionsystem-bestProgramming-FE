## How to Add Images

1. Copy all your pet images from your downloads/assets folder
2. Paste them into this `frontend/public/assets/` folder
3. Make sure the file names match exactly as listed above
4. Restart your frontend server if it's running

The images will then be accessible at `/assets/[imagename].png` in the application.
